# MatPreML 软件使用说明书

## 1. 软件简介

MatPreML (Materials Prediction via Machine Learning) 是一款基于机器学习的材料性质预测软件。该软件能够通过机器学习算法对材料的性能进行预测，帮助材料科学研究人员快速评估材料的潜在性能，加速新材料的研发过程。

## 2. 主要功能

- **多种机器学习模型支持**：支持包括线性回归、决策树、随机森林、支持向量机、梯度提升等多种机器学习算法
- **数据导入与处理**：支持CSV格式数据导入，自动处理数据特征
- **模型训练与优化**：提供贝叶斯超参数优化功能，提升模型预测精度
- **可视化分析**：提供丰富的图表展示功能，直观显示预测结果和模型性能
- **特征工程**：支持特征相关性分析和特征重要性评估
- **模型保存与加载**：可保存训练好的模型，便于后续使用

## 3. 系统要求

- 操作系统：Windows 7/8/10/11
- 内存：4GB RAM (推荐8GB或以上)
- 硬盘空间：至少500MB可用空间

## 4. 安装运行指南
1. 下载MatPreML软件包
2. 运行程序：   MatPreML.exe
  

## 5. 软件界面说明

### 5.1 主界面布局

主界面分为左右两个区域：
- **左侧控制面板**：包含数据导入、模型选择、训练、预测等功能按钮
- **右侧绘图区域**：显示数据可视化图表和模型预测结果

### 5.2 功能按钮说明

1. **导入学习数据**：导入用于训练模型的CSV格式数据文件
2. **选择特征值(X变量)画图**：选择要用于绘图的特征变量
3. **选择模型**：从下拉菜单中选择机器学习模型
4. **超参调优**：启用贝叶斯优化进行超参数调优
5. **训练 (单个模型)**：训练当前选择的模型
6. **训练 (所有模型)**：训练所有可用的模型并比较性能
7. **预测**：使用训练好的模型进行预测
8. **特征相关性**：分析特征之间的相关性
9. **特征重要性**：评估各特征对模型预测的重要性
10. **停止训练**：中断正在进行的模型训练过程

## 6. 使用流程

### 6.1 创建新项目

1. 点击菜单栏"文件" → "新建项目"
2. 输入项目名称和选择项目保存位置
3. 点击"创建"按钮

### 6.2 导入数据

1. 点击"导入学习数据"按钮
2. 选择CSV格式的数据文件
3. 软件将自动加载并显示数据

### 6.3 模型训练

1. 从"选择模型"下拉菜单中选择合适的机器学习模型
2. (可选)勾选"超参调优"以启用贝叶斯优化
3. 点击"训练 (单个模型)"或"训练 (所有模型)"按钮
4. 等待训练完成，查看模型评估指标

### 6.4 模型预测

1. 确保已有训练好的模型
2. 准备预测数据文件
3. 点击"预测"按钮
4. 选择预测数据文件
5. 查看预测结果

## 7. 高级功能

### 7.1 贝叶斯超参数优化

通过贝叶斯优化算法自动调整模型超参数，以获得更好的预测性能：
1. 在"设置"菜单中配置贝叶斯优化参数
2. 在训练前勾选"超参调优"选项
3. 开始模型训练

### 7.2 特征分析

- **特征相关性分析**：了解不同特征之间的相关程度
- **特征重要性评估**：识别对模型预测最重要的特征

## 8. 常见问题与解决方案

1. **无法导入数据**：检查数据文件格式是否为CSV，确保数据格式正确
2. **训练过程缓慢**：可调整CPU并行设置或减少贝叶斯优化迭代次数
3. **预测结果不准确**：尝试使用不同的模型或启用超参数优化
