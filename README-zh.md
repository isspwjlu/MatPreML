# MatPreML — 材料性能机器学习预测系统

MatPreML（Materials Prediction via Machine Learning）是一款基于机器学习的材料性能预测桌面软件。通过多种机器学习算法对材料的性能进行预测，帮助材料科学研究人员快速评估材料的潜在性能，加速新材料的研发进程。

![主界面截图](demo-1.png)
![训练结果截图](demo-2.png)

## 主要功能

- **多种机器学习模型**：支持线性回归、决策树、随机森林、SVR、梯度提升、Lasso、贝叶斯岭回归、神经网络等多种算法
- **数据导入与处理**：支持 CSV 格式数据导入，自动检测特征并进行可视化展示
- **模型训练与优化**：支持单个模型训练或一键训练全部模型，可选贝叶斯超参数优化
- **可视化分析**：丰富的图表展示预测结果和模型性能对比
- **特征工程**：特征相关性分析和特征重要性评估
- **模型持久化**：保存训练好的模型到文件，支持后续加载和预测
- **AI 助手集成**：内置 DeepSeek API 接口，支持智能数据分析
- **化学式解析**：自动提取化学式中的元素特征并进行编码

## 系统要求

- **操作系统**：Windows 7/8/10/11
- **内存**：4 GB（建议 8 GB 或以上）
- **硬盘空间**：500 MB 可用空间
- **Python**：3.8 或以上

## 快速开始

1. **下载或克隆** 本仓库
2. **安装依赖**：
   ```bash
   pip install -r requirements.txt
   ```
3. **启动程序**：
   ```bash
   python start.py
   ```
   或在 Windows 下直接双击 `start.cmd`。

## 使用说明

### 1. 创建新项目
   - 点击菜单 **文件 → 新建项目**，输入项目名称并选择保存位置。

### 2. 导入学习数据
   - 点击 **导入学习数据** 按钮，选择 CSV 格式的数据文件，软件将自动加载并显示数据。

### 3. 选择模型
   - 从下拉菜单中选择合适的机器学习模型。
   - （可选）勾选 **超参调优** 启用贝叶斯优化。

### 4. 训练模型
   - 点击 **训练（单个模型）** 或 **训练（所有模型）** 开始训练。
   - 训练完成后查看评估指标和预测图表。

### 5. 进行预测
   - 确保已有训练好的模型，点击 **预测** 按钮，选择预测数据文件，查看预测结果。

### 6. 特征分析
   - **特征相关性**：分析不同特征之间的相关程度。
   - **特征重要性**：评估各特征对模型预测的重要性。
